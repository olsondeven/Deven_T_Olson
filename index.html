MUSIC API

No Server Project

Hosted website: Muisc API
Github Repo: Muisc API Repo

This is a no server project that utilizes Spotify's api to search artists, albums, and tracks. The project also includes a midi controller. The midi controller is activated by clicking the midi button on the side menu. Once the midi controller is activated, you will be able to press the midi buttons allowing you to DJ on any page. In addition to the midi page, you are able to use your number key pad as the controller. Keys 0-9 have pre-set sounds. Data persists only due to local storage. Please do not use a real username or password due to all data being stored in local storage. If you forget your username or login, inspect application then go to local storage and it will show the current user and the corresponding password.

LOGIN

You will need to login to be able to save favorite artists, albums, or tracks. If you haven't created a username and password, you will need to go to the Create User page. Once created, log in and then you will be directed to your Favorites page.

CREATE

Please do not use real usernames or passwords. This is a no server project, to make data persist I have used local storage. Type in a username and password. Example username: JohnDoe565 Password: world565 Confirm Password: world565

FAVORITES

To view liked artists, albums, and tracks, you will need to press one of the three buttons on the top of the page. If you haven't liked anything, you should head over to the search page. Press the search button on the side menu and this will direct you to the search page. If you want to remove an item from your favorites, there is a minus button on each item. Click the minus button and it will remove the item from your favorites list.

SEARCH

There are three input boxes at the top of the page. The first input box is for searching artists, the second is for searching albums, and the last input is for tracks. Spotify only allows 30 second clips of their tracks to be requested. There will be a thumbs up button on each artist, album, or track. Press this if you would like to add the item to your favorites list. Once added to your favorites list, you can view them by pressing the favorites button on the side menu. There are previous and next page buttons at the top right of the page.

MIDI

Press the midi button on the side bar. This will do two things; first it will direct you to the midi page, then it will activate your number key pad as a midi controller. Once your side number key pad is activated, you will be able to use it on any page. If you do not have a side number key pad, you are able to use the midi controller on the midi page. Midi controller allows you to DJ to tracks or to make your own music.

Paul Valentine Clone

Full Webstack Application

Hosted website: Paul Valentine Clone
Github Repo: Paul Valentine Repo
Real website: Paul Valentine

This website is meant to be a clone of the real Paul Valentine website. Paul Valentine manufactures watches for women. This was a group project between three contributors and myself. Our group decided to clone the Paul Valentine site because it has won awards for outstanding design. The tech used in this project consisted of Javascript, JQuery, SCSS, CSS, ES2015, NodeJS, MongoDB, GoogleMaps, NPM, and AngularJS.

Shop Watches

The clone site allows you to view all products made by Paul Valentine. The watches that are available can be added to the cart. The cart persists through sessions. This was implemented on the backend through NodeJS. To access the cart click the button on the side menu at the bottom.

CART

To edit your cart, click the pay now securely button at the bottom of the right side menu.

Press the return to cart link(highlighted by green) in the middle of the page. This will direct you to a page were you can manipulate your cart data.

Now you will be able to delete items, add to cart, or remove all items.

INFO

On the home page in the middle of the view, click the journey button to view more info. This will bring you to the journal page where you can select other pages from the top nav bar.

Journal

Click any of the buttons on the top nav bar to go to other info pages. The footer at the bottom also has links to other views. Please feel free to explore.

MAGPUL CLONE

Full Stack Web Application

Hosted website:(Can demonstrate in person)
Github Repo: MAGPUL Clone Repo

This is a Full Web Stack project which is a clone of MAGPUL magpul.com. Magpul is a gun accessories provider. They make anything from gun stocks and magazines to apparel. MAGPUL was a fun website to clone due to their refreshing design. I have only cloned portions of the website which will be outlined in this walkthrough. Magazines are what MAGPUL is know for. I have created a SQL database using their magazine products. There is also a cart that products can be placed in. The tech used in this project consisted of Javascript, JQuery, SCSS, CSS, ES2015, NodeJS, SQL, NPM, and AngularJS. I have not hosted this website yet, but would be more than willing to walk you through it.

Landing Page

At the top of the landing page is a picture carousel. This carousel is implemented through the use of bootstrap. The rest of the site does not use bootstrap. Below the carousel are links to products or information. At the bottom of the landing page there are links to videos hosted by MAGPUL and a photo.

DROP DOWN MENU

At the top of every page there is a menu bar. This is created through a directive(simlar to a component). The drop down menu can be accessed through clicking products and PMAG Magazines. You will then be routed to the PMAG Magazines page.

PRODUCTS PAGE

On this page you are able to view all the magazines that MAGPUL offers. New magazines will be labeled with a new tag near the top left corner of the product. To view more information on the product, click the price button. This button will route you to the indiviual product page.

INDIVIDUAL PRODUCT PAGE

On the individual product page, you will be able to view the colors that are available for each product. To place the product in the cart, the user must select a color of their choosing. Next, click the add to cart button at the bottom right of the view. Once pressed, this will add the data to the cart. To access the cart the user will need to press the cart button on the menu bar. This is located at the top right of every view.

CART

The cart view allows you to manipulate the data stored in the cart. Users are able to subtract or add from the cart page.
